# Macro-tracker

This program is a macroeconomic data scorer.Currently the fundamental data is hardcoded, a future version will allow live user input per currency. It scores the values of the keys within the dictionary and gives the user the currency verdict neatly summarised at the bottom. 

What I put into practce:
Functions, loops and dictionaries combined into one functional program. I struggled to combine all three but with effort to learn them individually each day I managed to combine all three with ease.
